FTC Web Developer

The brief - to create a website for a Front End Web Developer traineeship in Finglas Training Centre. The client wanted the website to stand apart from the Finglas training Centre (FTC) site with its own domain name. This new website includes details of course content in HTML, CSS, and Javascript.

There was no information on certification on the FTC website, so a page was required to outline the components of the programme that need to be completed in order to gain certification from Quality and Qualifications Ireland (QQI).

It would have been preferable to add functionality to enable prospective students to apply online but the systems are not in place at FTC yet. A person can register their interest by applying through an external hub and be put on a waiting list or they can express an interest directly to FTC.
